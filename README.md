# Raspberry Pi 400 Labs

Coleção de laboratórios práticos desenvolvidos na Raspberry Pi 400 utilizando o Raspberry Pi OS Lite, com foco em administração de sistemas Linux, automação, acesso remoto e utilização da Raspberry Pi como plataforma para estudos, desenvolvimento e infraestrutura.

Os laboratórios são organizados de forma progressiva, partindo da preparação inicial do sistema até a configuração de um ambiente de desenvolvimento totalmente funcional. Cada laboratório documenta um conjunto específico de procedimentos, facilitando sua reprodução e servindo como material de consulta.

---

## Tecnologias utilizadas

Os laboratórios utilizam principalmente ferramentas do ecossistema Linux:

Raspberry Pi OS Lite (64-bit)
Raspberry Pi 400
OpenSSH
Bash
tmux
systemd
Git
Python
Ferramentas nativas do Linux (ip, journalctl, evtest, etc.)

---

## Laboratórios disponíveis

### Lab 1 – Primeira inicialização da Raspberry Pi 400

Preparação inicial da Raspberry Pi utilizando o Raspberry Pi OS Lite, atualização do sistema, primeiros comandos no Linux, organização do ambiente de trabalho e testes básicos do hardware em modo headless.

### Lab 2 – Preparação do ambiente headless/híbrido na Raspberry Pi 400

Configuração de um ambiente híbrido que combina acesso remoto via SSH com utilização local do teclado integrado da Raspberry Pi 400. Inclui configuração de aliases, tmux, testes do teclado em modo texto e ajustes para tornar a administração remota mais confortável.
